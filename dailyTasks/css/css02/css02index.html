Selectors element = HTML element

 Universal: (*), selects every element

 Type: (element)

 ID: (#(element ID))

 Class: (.(element class))

Group Selectors

 General: seperate with commas

 Combinators

  Descedent: descendant element

  Child: (>), strict descendant element

  Sibling

   Adjacent: (+), the element and the follwing one

   General: (~), the element and all subsequent ones

 Attribute

  One Attribute, regardless of value: (element[attr])

  One Attribute, in regards to value: (element[attr="value"])

  Multiple Attributes: (HTML [attr="value"][attr="value"]...)

 Pseudo-Class

  Dynamic: (element:pseudo class)

  Structural

   Child Elements: (element:nth-child(#))

   Particular Child: (element:nth-of-type(#)), (element:nth-child(#)), (element:first-of-type()), (element:last-of-type()), (element:only-of-type())

  Target: (:target)

  UI Elements States: (:enabled), (:disabled), (:checked)

  Language: (:lang(__))

  Negation: (:not(argument))

 Pseudo-Elements

  First Line: (element:first-line)

  First Letter: (element:first-letter)

  Content Before: (element:before)

  Content After: (element:after)

!important: used after any property value, style will be absolutely applied
