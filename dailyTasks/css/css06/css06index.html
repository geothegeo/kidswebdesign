Flexible Box

Code:

Output:

Notes:

(The code here only works for firefox. The css code below is becoming non-standard)

box-align: the top/bottom alignment of the children

box-pack: the left/right alignment of the children

box-direction: the display order of the children

box-flex: specifies that an element stretches or shrinks with the box

box-flex-group: assigns the element to a flex group

box-lines: specifies the number of lines to place the children in

box-ordinal-group: the lower the number, the more precedence it has to be displayed

box-orient: the orientation of the box display
