Background Properties

Code:

Notes:

background: (all-in-one declaration)

background-attachment: specifies how to move the background relative to the viewport

background-clip: sets whether an element's background color or image extends underneath its border

background-color: sets the color of the background

background-image: sets one or more background images for an element

background-origin: sets the background positioning area

background-position: sets the starting position of a background image

background-repeat: sets if/how a background image will be repeated

background-size: specifies the size of the background images
