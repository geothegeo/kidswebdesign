Notes:

<form> contains all form elements

<fieldset> draws a box around related elements

<legend> defines the caption for the form (inside <fieldset>)

<button> creates a clickable button

<select> makes a drop-down list with <option>

<option> defines the options within <select> or <datalist>

<optgroup> groups the <select> tags

<textarea> creates a multiline textbox
