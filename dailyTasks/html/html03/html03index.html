HTML Structural Elements

3) Figure Elements

3.1) List

Code:

Output:

Notes:

<ol> ordered list

<ul> unordered list

<li> list item (inside <ol> or <ul>)

<dl> definition list

<dt> defition term (inside <dl>)

<dfn> defines instance of a term

<dd> definition description (inside <dl>)


3.2) Progress and Meter

Code:

Output:

Notes:

<progress> defines progress

<meter> defines a scalar measurement


3.3) Table

Code:

Output:

Notes:

<tr> defines a table

<th> defines a table row (inside <table>)

<td> defines a table cell (inside <tr>)

<caption> defines a caption (immediately after <table>)

<col> specifies the properties for entire col (inside <colgroup>)

<colgroup> groups <col> (inside <table>)

<thead> optionally groups the <tr> tags, represents first row

<tfoot> optionally groups the <tr> tags, represents central rows

<tbody> optionally groups the <tr> tags, represents last row


3.4) Image

Code:

Output:

Notes:

<img> shows an image file

<map> defines an image-map that splits an image into clickable sections

<area> defines a part on an image-map that can be clicked (inside <map>)

<figure> can contain <img> and a <figcaption>

<figcaption> specifies a caption for <figure> (inside <figure>)


3.5) Multimedia

Code:

Output:

Notes:

<audio> plays a found file

<video> plays a movie file

<source> specifies alt. files (inside <audio> or <video>)

<track> specifies visible text when playing (inside <audio> or <video>)

<object> produces embedded object

<param> passes parameters to the plugin (inside <object>)

<embed> defines ext. app or plug-in

<canvas> reserves a region for a script to draw
